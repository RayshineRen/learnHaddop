- `hd_sim_cursor`：HDFS / MapReduce / YARN 模拟器（Notebook LM 提示词 + Cursor 实现）。
- `sp_sim_gpt/`：MiniSpark 教学演示（GPT 生成提示词 + Codex 实现）。
- `sp_sim_cursor`：MiniSpark 教学演示（GPT 生成提示词 + Cursor 实现）。
